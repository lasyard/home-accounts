## 1. Import 面板上设置列字段时 Crash

导入一个 CSV 文件，在 Import 面板上执行以下操作：

1. 设置某列的字段为 Amount
2. 设置这一列为 Invalid
3. 当再次试图设置这一列的字段时，**Crash**

## 2. Crash 时某些段被删除

当程序意外退出时，文件中原有的某些段内容被清空，如 `accounts` 段
