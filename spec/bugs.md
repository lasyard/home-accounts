# Bugs

这个文件只能由人类手动修改，AI agent 勿动。

以下复选框列表的每一项代表一项缺陷。每个缺陷前面有一个数字是它的编号，随后的文字为它的标题。每个缺陷的详细描述在此列表的后面的每个二级标题下面，编号与列表中的编号对应。

- [x] 1 Crash 时某些段被清空

当被要求修复某个缺陷时：

- 如果对应的复选框已选中，说明此缺陷已修复，应该忽略并报告“缺陷 XXXX 已修复”
- 修复缺陷时请先读 `/docs` 目录下面的文件以获取需要的信息
- 缺陷修复后只更新 `logs.md` 文件，不要更新此处的缺陷列表

## 1 Crash 时某些段被删除

当程序意外退出时，文件中原有的某些段内容被清空，如 `accounts` 段。
